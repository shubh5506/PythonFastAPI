CRUD API for Shipping Items (Python + FastAPI)

A lightweight RESTful CRUD API built with Python + FastAPI to manage shipping items (create, read, update, delete).
Perfect starter backend project to showcase API development skills.

### Features

# Create new shipping items
# Get all shipping items
# Get item by ID
# Update item details
# Delete an item
# Request validation using Pydantic
# Interactive API docs using Swagger UI (/docs)

### Tech Stack

#  Python 3.10+
#  FastAPI
#  Uvicorn
#  Pydantic
